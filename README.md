# k8-studio
K8 Studio is a Docker-first Kubernetes workbench for multi-cluster operations. Connect via kubeconfig or in-cluster service account, then manage Pods, Deployments, StatefulSets, PVCs, Services, Logs, Port-Forwards, env vars, and resource metrics from one web UI across Kubernetes, GKE, AKS, EKS, kind, and Docker Desktop.
